# Directory Service API

Микросервис для управления учреждениями, подразделениями, услугами и рабочими местами, которые используются при работе с электронной очередью.

## Функциональность

- **Учреждения (Facilities)**: CRUD операции для учреждений
- **Подразделения (Departments)**: Получение информации о подразделениях
- **Расписания (Schedules)**: Управление расписаниями работы подразделений
- **Услуги (Services)**: Каталог услуг подразделений
- **Рабочие места (Workplaces)**: Информация о рабочих местах в подразделении
- **Настройки очередей (Queues)**: Конфигурация электронных очередей
